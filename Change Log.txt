2007-10-26
发布SKYUC PHP V2.0,全新内核,全新界面,全新技术!
程序构造: PHP + MYSQL + Smarty + AJAX + JSON + XML + DIV + CSS + XHTML
SKYUC PHP是一款通用视频点平台构建软件，使用她您可以非常方便的开一个网上影院，在网上开展自己的影视娱乐。
SKYUC PHP有如下特点：
1、强大的模版机制 
SKYUC PHP使用PHP模板中功能最强大的Smarty结合最流行的 Adobe Dreamweaver 软件，采用DIV+CSS实现了一套模版机制，让您改动模版不再需要上传，而是在后台稍稍动动手设置一下就可以了。
2、功能的 AJAX 化
SKYUC PHP 使用目前流行的 AJAX 技术，为您实现简洁修改，奠定了基础。
同时在更多方面以减轻用户的劳动，提供工作的效率为宗旨，完整的实现了相关功能的 AJAX 化，由于是在底层完全支持，所以也为以后的扩展打下了基础。
3、高效率的代码和执行性能
当你浏览自己的网站，发现速度很慢，你首先想到的办法就是生成静态页面吗？
其实动态、静态页面在不同的用途下各擅胜场，但是在有很多个性化功能的前提下，动态的页面要比纯静态的页面好处多多，那么剩下就是一个效率问题。
我们认为，动态页面的瓶颈99％在数据库上，凭着天空网络开发组历经三年的数据库架构设计以及优化经验为基础，设计了目前的 SKYUC PHP 数据库结构，并通过缓存机制，实现目前的高效访问。
在不考虑网速的情况下，与纯静态页面相比，您不会感觉到丝毫的差别。我们也会在以后的开发设计中，让目前的架构更加完善、更加高效。
